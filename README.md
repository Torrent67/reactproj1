## Setup/Installation Requirements

1. Clone this repository:
    ```
    $ git clone 
    ```
2. Install dependencies:
    ```
    $ npm install
    ```
3. Build distribution files:
    ```
    $ webpack
    ```
4. Run local host server :
    ```
    $ npm run start
    ```
